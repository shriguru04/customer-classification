# Customer Segmentation & Prediction System

## 📌 Project Overview

This project is an end-to-end machine learning system designed to analyze customer behavior, segment users into meaningful groups, and predict customer categories for new data. It also includes anomaly detection to identify unusual customer patterns.

---

## 🌍 Real-World Impact

This system helps businesses better understand their customers by identifying distinct behavioral segments. It enables targeted marketing, personalized recommendations, improved customer retention strategies, and early detection of abnormal or fraudulent behavior. This leads to better decision-making, increased revenue, and improved customer experience.

---

## 🚀 Features

* Data preprocessing and missing value handling
* Feature encoding and scaling
* Anomaly detection using Isolation Forest
* Dimensionality reduction using PCA
* Optimal clustering using K-Means
* Customer segmentation analysis
* Supervised classification using Random Forest
* Prediction for new/unseen customers
* Model evaluation using cross-validation and metrics

---

## 🧠 Machine Learning Workflow

1. Data Cleaning & Imputation
2. Feature Encoding (Label Encoding)
3. Standardization (Scaling)
4. Anomaly Detection (Isolation Forest)
5. Dimensionality Reduction (PCA)
6. Optimal Cluster Selection (Silhouette Score + K-Means)
7. Model Training (Random Forest Classifier)
8. Evaluation (Accuracy, Precision, Recall, F1-score, Cross-validation)
9. Prediction on new customer data
